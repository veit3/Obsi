# Определение
Пусть $f(x, y)$ - определена и непрерывна на полосе $[a, +\infty] \times [c, d]$. Несобственным интегралом от ограниченной функции называют значение предела: $$F(y) = \lim_{b \to \infty} \int_a^bf(x, y)dx = \int_a^{+\infty}f(x, y)dx$$Аналогичным образом определяется интеграл, если $a \to -\infty$

Пусть теперь $f(x, y)$ - определена на множестве $[a, b) \times [c, d]$ (точка $b$ - особая точка, т.е. в этой точке $f(x, y)$ обращается в $\pm\infty$). Тогда несобственным интегралом от неограниченной функции называют значение предела $$F(y) = \lim_{\lambda\to0+}\int_a^{b-\lambda}f(x, y)dx = \int_a^bf(x, y)dx$$
# Равномерная сходимость
Говорят, что интеграл $\int_a^{+\infty}f(x, y)dx$ сходится равномерно, если $$\forall \epsilon>0\ \exists A_0(\epsilon)=A_0 \geq a : \forall A>A_0\implies \left|\int_A^{+\infty}f(x, y)\,dx\right| < \epsilon$$(Обратите внимание, что $A_0$ зависит только от $\epsilon$, а не от $y$)
# Критерий Коши
Интеграл $\int_a^{+\infty}f(x, y)dx$ сходится равномерно только и только тогда, когда $$\forall \epsilon>0\ \exists M > a: \forall A_1, A_2 > M \implies \left|\int_{A_1}^{A_2}f(x, y)dx\right| < \epsilon$$
# Доказательство
1. Необходимость $(\Rightarrow)$
Пусть $$F(y)=\int_a^{+\infty}f(x, y)\,dx$$Сходится равномерно по $y$. Тогда по определению для любого $\epsilon>0$ существует такое $M \geq a$, что для всех $A > M$ и любого $y \in [c, d]$ выполняется: $$\left|\int_A^{+\infty}f(x, y)\,dx\right|<\frac{\epsilon}{2}$$Выберем $A_1, A_2 > A$, тогда $$\left|\int_{A_1}^{A_2}f(x, y)\, dx\right| = \left|\int_{A_1}^{+\infty}f(x, y)\, dx-\int_{A_2}^{+\infty}f(x, y)\, dx\right|$$По неравенству треугольника $$\left|\int_{A_1}^{+\infty}f(x, y)\, dx-\int_{A_2}^{+\infty}f(x, y)\, dx\right| \leq \left|\int_{A_2}^{+\infty}f(x, y)\, dx\right| + \left|\int_{A_1}^{+\infty}f(x, y)\, dx\right| \leq \frac{\epsilon}{2}+\frac{\epsilon}{2} = \epsilon$$Необходимость доказана $\blacksquare$
