# Как склонировать репозиторий и вносить изменения в него
Чтобы склонировать репозиторий, установите git и пропишите:
```bash
# Первоначальная настрйока. Нужно проделать один раз
git config --global user.name "ВАШ ЖЕЛАЕМЫЙ НИК"
git config --global user.email "ВАШ EMAIL"
```

Затем клонируете репозиторий
```bash
cd C:\ПАПКА\КУДА\СКЛОНИРУЕТСЯ\РЕПОЗИТОРИЙ
git clone https://github.com/veit3/Obsi.git
```

Если же вы обнаружили ошибку, можете поправить её:
```bash
cd C:\ПАПКА\КУДА\СКЛОНИРУЕТСЯ\РЕПОЗИТОРИЙ # Если вы еще не находитесь в ней
git remote add origin https://github.com/veit3/Obsi.git # Пропишите один раз. Добавляет адрес репозитория в git
git fetch # Скачать последние изменения репозитория. Не обязательно, если он не менялся
git add .
git commit -m "СООБЩЕНИЕ. Желательно напишите, что конкрентно меняли"
git push
```
