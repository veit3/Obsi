# Мажорантный признак Вейерштрасса
Пусть $f(x, y)$ определено на полосе $[a, +\infty) \times [c, d]$. Если существует $g(x)$ такая, что
- $g(x)$ - определена на $[a, +\infty)$
- $|f(x, y)| \leq g(x)$
- Интеграл $\int_a^{+\infty}g(x)\,dx$ - сходится
Тогда $F(y) = \int_a^{+\infty}f(x, y)\,dx$ сходится равномерно и абсолютно

## Док-во:
Интеграл $\int_a^{\infty}g(x)\,dx$ сходится, это значит, что для любого $\epsilon>0$ существует такое $A_0>a$, что для всех $A>A_0$ выполняется: $$\int_A^{+\infty}g(x)\,dx < \epsilon$$Оценим хвост исходного интеграла $F(y)$: $$\left|\int_A^{+\infty}f(x, y)\,dx\right| \leq \int_a^{+\infty}|f(x, y)|\,dx \leq \int_B^{+\infty}g(x)\,dx < \epsilon$$Итак выбор $B_0$ зависит только от $\epsilon$, и не зависит от $y$, это значит, что $F(y)$ - сходится равномерно
Кроме того, $\int_a^{+\infty}|f(x, y)|\,dx \leq \int_B^{+\infty}g(x)\,dx < \epsilon$, это значит, что $F(y)$ - сходится абсолютно
